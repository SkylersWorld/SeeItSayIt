# Community Emergency Alert System (SeeItSayIt)

## Overview
a real-time, location-based emergency communication 
platform designed to bridge the gap between citizens and local authorities. 
It addresses the critical problem of slow local emergency communication by allowing users to 
report incidents such as fires, floods, or violence instantly.

## Key Features
* **Real-Time Alerts:** Powered by WebSockets for instant notification.
* **Location-Based Services:** Interactive map view for incident tracking.
* **Role-Based Access:** Distinct workflows for Citizens (reporting) and Authorities (verifying).
* **Priority Queueing:** Verified reports receive higher visibility.
* **Secure Authentication:** Email verification and optional anonymity for sensitive reports.

## Technology Stack
* **Backend:** Java (Spring Boot), WebSockets, Spring Security.
* **Frontend:** HTML5, CSS3, JavaScript (Leaflet.js for Maps).
